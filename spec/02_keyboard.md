Here is the shortcut mapping translating the classic AtariWriter 80 keystrokes into a modern, intuitive keyboard layout for Safari Writer.

This mapping replaces the proprietary Atari keys (`[Option]`, `[Select]`, `[Start]`) with standard modern equivalents (`Ctrl`, `Alt`, `Shift`, and dedicated navigation keys) while preserving the original functionality.

### Navigation & Cursor Movement

| Legacy Shortcut (Atari) | Modern Shortcut | What It Does |
| --- | --- | --- |
| <br>`[Select] [T]` 

 | **Ctrl + Home** | Moves the cursor instantly to the top of the file. |
| <br>`[Select] [B]` 

 | **Ctrl + End** | Moves the cursor instantly to the bottom of the file. |
| <br>`[Control] [A]` 

 | **Home** | Moves the cursor to the beginning of the current line. |
| <br>`[Control] [Z]` 

 | **End** | Moves the cursor to the end of the current line. |
| <br>`[Select] [Left/Right Arrow]` 

 | **Ctrl + Left/Right Arrow** | Skips the cursor to the beginning of the next or previous word. |
| <br>`[Option] [Up/Down Arrow]` 

 | **Page Up / Page Down** | Scrolls page by page through the text file. |

### Deletion & Basic Editing

| Legacy Shortcut (Atari) | Modern Shortcut | What It Does |
| --- | --- | --- |
| <br>`[Control] [Insert]` 

 | **Insert** | Toggles between Insert mode (pushes text) and Type-over mode. |
| <br>`[Shift] [Delete/Bk Sp]` 

 | **Shift + Delete** | Deletes text from the cursor to the end of the current screen line. |
| <br>`[Shift] [Insert]` 

 | **Ctrl + Z** | Restores the last deleted line (Undelete) at the cursor position. |
| <br>`[Control] [Caps]` 

 | **Shift + F3** | Toggles the character under the cursor between upper- and lowercase. |
| <br>`[Control] [P]` 

 | **Ctrl + M** | Marks the beginning of a paragraph for automatic indentation without a carriage return. |

### Block Operations (Cut, Copy, Paste)

*Note: The modern mapping shifts away from Atari's "anchor" system to the standard modern selection paradigm.*
| Legacy Shortcut (Atari) | Modern Shortcut | What It Does |
| :--- | :--- | :--- |
| `[Option] [B]` then `[Option] [Delete]`  | **Ctrl + X** | Cuts the selected block of text and stores it in the failsafe buffer (clipboard). |
| `[Option] [B]` then `[Option] [E]`  | **Ctrl + C** | Copies the selected block to the failsafe buffer without deleting it. |
| `[Option] [X]`  | **Ctrl + V** | Pastes the contents of the failsafe buffer at the cursor. |
| `[Select] [Delete/Bk Sp]`  | **Ctrl + Shift + Delete** | Deletes all text from the cursor to the end of the file. |
| `[Option] [A]`  | **Alt + A** | Alphabetizes a highlighted list of words. |
| `[Option] [W]`  | **Alt + W** | Counts the number of words in a highlighted block or the whole file. |

### Search & Replace

| Legacy Shortcut (Atari) | Modern Shortcut | What It Does |
| --- | --- | --- |
| <br>`[Start] [S]` 

 | **Ctrl + F** | Prompts for a search string and finds the first occurrence. |
| <br>`[Start] [R]` 

 | **Ctrl + H** | Prompts for a replacement string. |
| <br>`[Select] [S]` / `[Select] [R]` 

 | **F3 / Shift + F3** | Finds the next occurrence / Replaces the current occurrence. |
| <br>`[Option] [G]` 

 | **Alt + R** | Performs a global search and replace to the end of the file. |

### Inline Formatting & Special Functions

| Legacy Shortcut (Atari) | Modern Shortcut | What It Does |
| --- | --- | --- |
| <br>`[Select] [.]` 

 | **Ctrl + B** | Toggles bold print. |
| <br>`[Control] [U]` 

 | **Ctrl + U** | Toggles underlining. |
| <br>`[Control] [C]` 

 | **Ctrl + E** | Centers the current line of text. |
| <br>`[Control] [C]` x2 

 | **Ctrl + R** | Blocks the line of text flush right. |
| <br>`[Option] [P]` 

 | **Ctrl + P** | Launches Print Preview. |
| <br>`[Option] [Insert]` 

 | **Alt + F** | Inserts a "Form Printing" blank to be filled out during the print process. |
| <br>`[Option] [M]` 

 | **Alt + M** | Inserts a Mail Merge database character for generating form letters. |
