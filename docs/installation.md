# Installation

Safari Writer is built with Python and can be installed via standard Python package managers.

## Requirements

- **Python**: 3.10 or later.
- **Dependencies**: Textual, pyenchant, pygments (automatically installed).

## Install via pip

```bash
pip install safari-writer
```

## Install via uv (Recommended)

```bash
uv tool install safari-writer
```

## Running the Applications

Once installed, you can start each application directly from your command line:

- `safari-writer` — Start the word processor.
- `safari-dos` — Start the file manager.
- `safari-chat` — Start the help assistant.
