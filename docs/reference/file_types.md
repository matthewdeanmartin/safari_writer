# File Types

Safari Writer works with these file types:

- **.sfw** — Safari Writer formatted document (includes formatting codes)
- **.txt** — Plain text (no formatting)
- **.md** — Markdown files

Safari Writer can also open source code files with syntax highlighting: `.py`, `.js`, `.rs`, `.go`, `.java`, `.c`, `.cpp`, `.h`, `.sql` and others. These are read-only for reference.
