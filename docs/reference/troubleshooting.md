# Troubleshooting

Common questions and issues.

## Tips
- **The cursor is not where I expect it**: Check if you are in Insert or Type-over mode. The status bar shows which mode is active. Press **Insert** to toggle.
- **My formatting markers look strange**: This is normal. Formatting shows as visible markers in the editor (like ← for bold). They will render correctly when you print or export.
- **The proofreader flags correct words**: Add them to your personal dictionary. Press **W** in the Proofreader to save your accepted words.
- **Accidentally deleted a file**: Check Safari DOS Garbage. Press **G** from the Safari DOS menu. Your file may still be there.
- **I want to change the colors**: Press **X** from the Main Menu to open the Style Switcher.

## Common Issues
- **Application crashed**: Try restarting. If the problem persists, check whether your document file is very large or contains unusual characters.
- **Cannot find my file**: Use Safari DOS (press **O** from the Main Menu) to browse your folders. Press **/** to filter by name.
