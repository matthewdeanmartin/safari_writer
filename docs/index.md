# About Safari Writer

Safari Writer is a retro text-mode word processor inspired by the classic AtariWriter 80. Built with Python and the Textual framework, it brings back the focused, distraction-free environment of 8-bit computing to modern systems.

It is part of a suite of applications:

- **Safari Writer**: The core word processor.
- **Safari DOS**: A full-featured file manager.
- **Safari Chat**: An ELIZA-style help assistant.
- **Safari Fed**: A calm, keyboard-driven Mastodon client.
- **Safari Base**: A dBASE-style SQLite shell for address-book and table work.
- **Safari ASM**: An assembly-flavored interpreter for `.asm` programs.
- **Safari Basic**: The shared Atari BASIC-compatible macro engine.
- **Safari REPL**: A small Atari BASIC REPL and `.BAS` file runner.
- **Safari Reader**: A keyboard-first book reader with a local library and Gutenberg catalog.
- **Safari Slides**: A SlideMD presentation viewer.
- **Safari View**: A retro image viewer and renderer.

All of them share a consistent retro aesthetic and are designed for fast, keyboard-driven operation. No mouse required.

Use the module pages under `docs\modules\` when you want the exact user-facing commands and workflows for a specific tool.
