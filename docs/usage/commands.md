# Keyboard Commands

Safari Writer is designed for fast, keyboard-driven editing.

## Navigation
- **Arrow Keys** — Move one character or line.
- **Ctrl + Left/Right** — Jump word by word.
- **Home / End** — Jump to start or end of the current line.
- **Ctrl + Home / End** — Jump to top or bottom of the file.
- **Page Up / Page Down** — Scroll one page at a time.

## Selection & Editing
- **Shift + Navigation** — Select text.
- **Ctrl + X / C / V** — Cut, Copy, and Paste.
- **Backspace / Delete** — Delete characters or selections.
- **Ctrl + Z** — Undo the last action.

## Search and Replace
- **Ctrl + F** — Find text.
- **F3** — Find the next occurrence.
- **Alt + H** — Set the replacement string.
- **Alt + N** — Replace current match and find next.
- **Alt + R** — Replace all occurrences.

## Formatting Codes
These codes appear as visible markers in the editor.
- **Ctrl + B** — Bold (← marker)
- **Ctrl + U** — Underline (▄ marker)
- **Ctrl + G** — Elongated (E marker)
- **Ctrl + [** — Superscript (↑ marker)
- **Ctrl + ]** — Subscript (↓ marker)
- **Ctrl + E** — Center (↔ marker)
- **Ctrl + R** — Flush Right (→→ marker)

## Document Structure
- **Ctrl + Shift + H** — Insert Header line.
- **Ctrl + Shift + F** — Insert Footer line.
- **Ctrl + Shift + S** — Insert Section heading.
- **Ctrl + Shift + E** — Insert Page Break.
