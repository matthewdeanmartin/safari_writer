# Quick Start

Get up and running with Safari Writer.

## 1. Start Safari Writer
Run `safari-writer` from your command line to open the Main Menu.

## 2. Load the Demo
Press **T** from the Main Menu. This loads a demo document that shows you what formatted text looks like in the editor.

## 3. Creating and Editing
- **C** — Create a new file.
- **E** — Edit the current document.
- **L** — Load a file from disk.
- **S** — Save your progress.

## 4. Basic Editor Use
- **Insert** — Toggle between Insert and Type-over mode.
- **Enter** — Start a new paragraph.
- **Tab** — Move to the next tab stop.
- **Escape** — Return to the Main Menu.

## 5. View your work
Press **P** to see print and export options. Choose **A** for an ANSI Preview to see your formatted document right in the terminal.
