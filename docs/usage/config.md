# Configuration

Customize your writing environment and document layout.

## Global Format

Press **G** from the Main Menu to set margins, spacing, and other layout options.

- **T/B/L/R** — Top, Bottom, Left, and Right margins.
- **S** — Line Spacing (2=single, 4=double, 6=triple).
- **D** — Paragraph Spacing.
- **I** — Paragraph Indentation.
- **J** — Justification (0=ragged, 1=justified).

## Style Switcher

Press **X** from the Main Menu to choose a color theme.

- **Up / Down** — Browse available themes.
- **Enter** — Apply and save the theme.
- **Escape** — Cancel and keep the current theme.

Your theme choice persists across sessions.
