# Workflows

Safari Writer includes powerful tools for proofreading, mail merging, and printing.

## 1. Proofreader
Press **V** from the Main Menu to verify your spelling.
- **H** — Highlight misspelled words.
- **C** — Correct errors by stopping at each one.
- **S** — Dictionary Search.
- **W** — Write to your Personal Dictionary.

## 2. Mail Merge
Press **M** from the Main Menu for database and form letters.
- **C** — Create a new database.
- **F** — Format records with up to 15 fields.
- **E** — Enter and edit records.
- **Alt + M** — Insert a merge field (@ marker) in your document.
- **P** — Print your document for each record in the database.

## 3. Printing and Exporting
Press **P** from the Main Menu or **Ctrl + P** in the editor.
- **ANSI Preview** — See how your document looks with margins and formatting.
- **Export to Markdown** — Save as a `.md` file for modern use.
- **Export to PostScript** — Save as a `.ps` file for high-quality printing.
