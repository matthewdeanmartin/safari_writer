# Safari Slides

Safari Slides is a keyboard-first presentation viewer for SlideMD decks.

## Starting Safari Slides

```bash
safari-slides
safari-slides docs\safari_suite_tools.slides.md
```

If you do not pass a deck, Safari Slides opens its built-in welcome presentation.

## Navigation

- **Space / Right / Down / PageDown** — Next slide or fragment.
- **Left / Up / PageUp** — Previous slide or fragment.
- **Home** — First slide.
- **End** — Last slide.
- **N** — Toggle speaker notes.
- **Q / Esc** — Quit the viewer.

## What Safari Slides Shows

- The current deck title.
- Slide number and layout information in the header.
- Slide notes when notes are enabled.
- Fragment progress for slides that reveal content in steps.

## SlideMD Format

Safari Slides reads Markdown-style decks, including `.slides.md` and `.slidemd` files, with `---` used to separate slides.
