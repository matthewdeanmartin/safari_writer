# Safari Reader

Safari Reader is a keyboard-first terminal reading app with a local library, an online Project Gutenberg catalog, bookmarks, and adjustable reading settings.

## Starting Safari Reader

```bash
safari-reader
safari-reader --library C:\books
```

## Main Menu

- **R** — Resume the last book you opened.
- **L** — Open your library.
- **C** — Open the online catalog.
- **H** — Show quick help.
- **S** — Open reader settings.
- **Q / Esc** — Quit.

## Library and Catalog

In the library:

- **R / Enter** — Read the selected book.
- **I** — Import a local file.
- **D** — Show book details.
- **A** — Archive a book.
- **C** — Jump to the online catalog.

In the catalog:

- **S** — Search Project Gutenberg.
- **T** — Show top downloads.
- **Enter** — View details for the selected book.
- **D** — Download the selected book.

## Reading Controls

- **PgDn / Space** — Next page.
- **PgUp** — Previous page.
- **Ctrl + PgDn / Ctrl + PgUp** — Next or previous chapter.
- **Home / End** — Start or end of the current chapter.
- **/** — Search inside the current book.
- **B** — Set a bookmark.
- **G** — Go to a percent, chapter, or bookmark.
- **T** — Toggle the table of contents.
- **+ / =** — Bigger text.
- **-** — Smaller text.
- **] / [** — More or less line spacing.
- **H** — Open help.
- **L** — Return to the library.
- **Q / Esc** — Back or quit.

## Settings

Safari Reader settings let you choose text scale, spacing, margins, page mode, and theme from the keyboard.
