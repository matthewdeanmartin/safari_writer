# Safari Writer

Safari Writer is the core word processor in the Safari suite. It is the main shell most people start in, and it is also the place where print/export, proofreading, mail merge, and companion tools come together.

## Starting Safari Writer

```bash
safari-writer
```

The main menu is grouped into three columns:

- **Words** — Create, edit, proofread, print, global format, mail merge, library reader, and diagnostics.
- **DOS** — File browsing, backup/restore, load, save, save as, delete, new folder, and quit.
- **Tools** — Safari DOS, Safari Base, Safari Chat, Safari Fed, Safari Slides, Safari REPL, Safari View, Style Switcher, and Demo Mode.

## Editor Basics

- **Insert** — Toggle insert/type-over mode.
- **Enter** — Start a new line.
- **Tab** — Move to the next tab stop.
- **Esc** — Return to the main menu.
- **Ctrl + F** — Find text.
- **F3** — Find next match.
- **Alt + H / Alt + N / Alt + R** — Set replace text, replace next, or replace all.

## Writing Tools

- **V** — Proofreader.
- **P** — Print and export.
- **G** — Global Format for margins, tabs, spacing, and layout settings.
- **M** — Mail Merge.
- **Alt + W** — Word count.
- **Alt + A** — Alphabetize selected lines.

## Formatting

Safari Writer uses visible formatting markers inside the editor and applies them during print/export.

- **Ctrl + B** — Bold.
- **Ctrl + U** — Underline.
- **Ctrl + G** — Elongated / double-width text.
- **Ctrl + E** — Center the current line.
- **Ctrl + R** — Flush right.
- **Ctrl + Shift + H / F / S / E / C** — Header, footer, section heading, page break, or chained print file.

## Suite Handoff

Safari Writer can launch companion tools directly from the main menu, including Safari DOS, Safari Chat, Safari Fed, Safari Base, Safari Reader, Safari REPL, Safari Slides, and Safari View.
