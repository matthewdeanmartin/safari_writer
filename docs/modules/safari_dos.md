# Safari DOS

Safari DOS is a full-featured file manager integrated with Safari Writer.

## Main Menu
- **F** — Browse your file list.
- **D** — Select different devices and volumes.
- **G** — Browse the Garbage folder for deleted files.
- **H / F1** — Open the DOS help screen.
- **Y** — Style Switcher to change the DOS color theme.
- **Q / Esc** — Quit Safari DOS.

## File Browser
Manage your files with these keyboard shortcuts:
- **Up / Down** — Move the cursor.
- **Home / End** — Jump to the first or last item.
- **PageUp / PageDown** — Scroll by a page.
- **Enter** — Open the selected file or folder.
- **Backspace** — Go to the parent folder.
- **E** — Run the selected program file.
- **H** — Jump to the home folder.
- **.** — Toggle hidden files.
- **/** — Filter the listing by pattern.
- **S** — Cycle sort order.
- **D** — Switch devices.
- **C** — Copy selected files.
- **M** — Move selected files.
- **R** — Rename a file.
- **W** — Duplicate a file.
- **N** — Create a new folder.
- **X** — Move files to Garbage (recoverable).
- **I** — File information (size, date, permissions).
- **P** — Toggle read-only protection.
- **Z** — Create a zip archive from the selection.
- **U** — Unzip the selected archive.
- **V** — Toggle the preview pane.
- **Space** — Open the fullscreen preview.
- **F** — Open Favorites / recent locations.
- **G** — Open the Garbage listing.

## Selection and picker mode
- **T** — Toggle the current item selection.
- **A** — Select all items.
- **Tab** — Choose the current folder when DOS is being used as a folder picker.

## Garbage Recovery
Deleted files stay in the Garbage folder until you manually empty it or restore them.
- **Enter** — Restore the selected file.
- **Delete** — Permanently remove the file.

## Favorites
Bookmark frequently-used folders for quick access.
- **F** — Toggle the current folder as a favorite.
- **Enter** — Open the selected favorite.

Safari DOS can run `.bas`, `.asm`, `.prg`, and `.py` files directly from the browser.
