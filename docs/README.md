# Safari Writer Docs

Safari Writer is a retro text-mode word processor and tool suite built with Python and Textual. The docs cover the editor itself plus the companion apps that appear in the Safari Writer main menu.

## Suite modules

- `docs\modules\safari_writer.md` — the core word processor, print/export flow, proofreader, and mail merge entry points
- `docs\modules\safari_dos.md` — file browser, favorites, garbage, and file operations
- `docs\modules\safari_chat.md` — ELIZA-style help assistant and its slash commands
- `docs\modules\safari_fed.md` — keyboard-driven Mastodon client
- `docs\modules\safari_base.md` — dBASE-style SQLite shell
- `docs\modules\safari_basic.md` — BASIC macro engine used by Writer and Fed
- `docs\modules\safari_repl.md` — interactive BASIC workspace and file runner
- `docs\modules\safari_reader.md` — library, catalog, and focused reading workflows
- `docs\modules\safari_slides.md` — SlideMD presentation viewer
- `docs\modules\safari_view.md` — retro image viewer and renderer

## Task guides
